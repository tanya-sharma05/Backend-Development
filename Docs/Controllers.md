# Controllers

Controllers are the business logic layer of your application. They handle incoming requests, process data, interact with models/database and send responses back to the client.

## Purpose
- Contain the core business logic
- Process data from requests
- Interact with database/models
- Return appropriate responses
- Keep routes clean and focused

## Key Responsibilities
1. **Request Processing**: Extract and validate data from req object
2. **Business Logic**: Implement the core functionality
3. **Data Operations**: CRUD operations with database
4. **Response Handling**: Send appropriate HTTP responses
5. **Error Handling**: Catch and handle errors
