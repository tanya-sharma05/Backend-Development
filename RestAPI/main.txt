REST API -> JSON

1. GET /users -> Render HTML doc
   GET /api/users -> List all users in JSON format

2. GET /users/1 -> Get the user with id 1

3. POST /users -> Create a new user

4. PUT /users/1 -> Update the user with id 1

5. PATCH /users/1 -> Partially update the user with id 1

6. DELETE /users/1 -> Delete the user with id 1

POSTMAN is a utility tool for api testing and documentation
